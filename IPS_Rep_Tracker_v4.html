Database setup
Connect IPS Rep Tracker to your Supabase database.
1
Go to supabase.com → your IPS-Tracker project → SQL Editor → New query → paste & run:
-- Run once to set up / update tables ALTER TABLE public.customers ADD COLUMN IF NOT EXISTS paint_brand text default ''; ALTER TABLE public.customers ADD COLUMN IF NOT EXISTS main_supplier text default ''; ALTER TABLE public.visits ADD COLUMN IF NOT EXISTS actions text default ''; CREATE TABLE IF NOT EXISTS public.pins ( rep text primary key, pin text not null ); ALTER TABLE public.pins ENABLE ROW LEVEL SECURITY; DROP POLICY IF EXISTS "open" ON public.pins; CREATE POLICY "open" ON public.pins FOR ALL USING (true) WITH CHECK (true);
2
Get your credentials. Project Settings → API → copy Project URL and anon key.
