First-time setup
Connect IPS Rep Tracker to a free Supabase database so all your reps sync in real time. Takes about 5 minutes.
1
Create a free account at supabase.com → New project → name it IPS-Tracker
2
Create the tables. Go to SQL Editor → New query → paste & run:
create table public.customers ( id bigserial primary key, name text not null, rep text not null default '', created_at timestamptz default now() ); create table public.visits ( id bigserial primary key, rep text not null, customer text not null, visit_date date not null, visit_type text default 'In-person', has_order boolean default false, issues text default '', notes text default '', next_visit date, score integer, created_at timestamptz default now() ); alter table public.customers enable row level security; alter table public.visits enable row level security; create policy "open" on public.customers for all using (true) with check (true); create policy "open" on public.visits for all using (true) with check (true);
3
Get your credentials. Project Settings → API → copy the Project URL and anon/public key.
🔔 Push Notifications (optional)
Get a phone notification every time a rep logs a visit. Install the free ntfy app on your phone (Android & iOS), then subscribe to your topic name below.
In the ntfy app: tap + → enter this topic name → subscribe. That's it.
